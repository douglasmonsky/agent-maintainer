# Measured Case Studies

These case studies are small, reproducible fixtures that show how Agent
Maintainer behaves during an agent repair loop. They are not marketing
benchmarks and they do not claim production-scale performance.

Each case study records:

- fixture repository used;
- commands run;
- measured result;
- what the agent should learn from the evidence;
- limitations of the fixture.
