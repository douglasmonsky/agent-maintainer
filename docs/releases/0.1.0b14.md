## Candidate Highlights

- Adds explicit TypeScript package-manager audit configuration for npm, pnpm,
  Yarn, and Bun without inferring a manager or executing unreviewed commands.
- Normalizes bounded JSON/NDJSON audit facts, failure records, summaries, and
  context across configured TypeScript workspaces with safe-path handling and
  deterministic ordering.
- Includes pinned npm and pnpm public projections plus malformed-input,
  clean-report, retention, and transport coverage; Yarn and Bun remain
  fixture-only until equivalent public evidence is collected.
- Keeps audit output advisory-only: vulnerability findings do not create a
  blocking security or reviewability gate.

The public configuration additions require the `0.1.0b14` prerelease bump and
keep contract revision 1. TypeScript/JavaScript remains experimental.
